💡 あなうめ九九 (AnauMe Kuku)
「あなうめ九九」は、掛け算（九九）の「答え」ではなく、「掛ける数」や「掛けられる数」を答えるトレーニングができるWebアプリケーションです。

答えを導き出すだけでなく、式全体を俯瞰して数字の関係性を掴む「逆思考」の力を養います。

🌟 特徴
虫食い（穴埋め）問題に特化

□ × 2 = ? (左側)

2 × □ = ? (右側)

ランダム出題

上記3パターンから選択可能です。

2つの回答モード

テンキーモード: 画面上の数字キーで答えを入力するモード。

フラッシュモード: 入力不要。頭の中で答えてからタップして正解を確認するモード（操作負担を減らし、回転率を上げたい場合に最適）。

苦手特訓機能

間違えた問題を自動で記録し、「苦手特訓」モードで重点的に復習できます。

保護者用ダッシュボード

学習履歴、正答率、解答タイム、間違えた問題の履歴を詳細に確認できます。

視覚・聴覚フィードバック

正解・不正解の効果音や、視覚的なアニメーション（揺れる、ポップするなど）で、子供のモチベーションを維持します。

🚀 使い方
1. アプリの起動
index.html (または保存したHTMLファイル) をGoogle ChromeやSafariなどのモダンブラウザで開いてください。 ※ 初回読み込み時にライブラリを取得するため、インターネット接続が必要です。

2. メニュー画面
あなうめの場所: 「□ × 2」「2 × □」「どっちも」から選択します。

問題数: 「10もん」「20もん」「50もん」から選択します。

苦手特訓: 過去に間違えたデータがある場合、ここから復習ができます。

モード選択: 画面下部で「テンキー（入力）」か「フラッシュ（見るだけ）」を切り替えます。

スタート: オレンジ色のボタンを押して開始します。

3. ゲーム画面
テンキーモード: 答えを入力してください。1桁入力した時点で即座に判定されます。

フラッシュモード: 答えを思い浮かべてから、画面全体をどこでも良いのでタップしてください。正解が表示されます。

4. 保護者メニュー（記録）
メニュー画面または結果画面の「記録（Userアイコン）」ボタンからアクセスできます。

日時の履歴、その時の正答率、かかった時間などを確認できます。

「全削除」で履歴をリセットできます。

🛠 技術スタック
ビルド不要のシングルファイル構成です。

HTML5

React 18 (CDN経由)

Babel (ブラウザ内トランスパイル)

Tailwind CSS (スタイリング)

📦 インストール・実行方法
このアプリケーションはサーバーへのデプロイやNode.js環境を必要としません。

ソースコードをテキストエディタに貼り付け、index.html として保存します。

保存したファイルをダブルクリックしてブラウザで開くだけで動作します。

⚠️ 注意事項
インターネット接続: ReactやTailwind CSSのライブラリをCDNから読み込んでいるため、オフラインでは動作しません（キャッシュが効いている場合を除く）。

データ保存: 学習履歴はブラウザの localStorage に保存されます。ブラウザのキャッシュをクリアすると履歴も消えますのでご注意ください。
